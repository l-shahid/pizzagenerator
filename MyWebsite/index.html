Random Pizza Generator Google Chrome Extension

This project started out when a friend and I could not decide what sort of pizza to order from Dominos. I joked that we should build a pizza randomizer that has capability to order a random pizza automatically for you from dominos.com. From there I decided to go ahead and start working on this project. Knowing that it would be a web application, I decided to work on the randomizer script in JavaScript. After the script had been tested and confirmed working I went ahead and worked on designing an HTML page for the randomizer. Once that was completed, I went ahead and started integrating it into a Google Chrome extension. All the project files can be found on my GitHub account under the "randomPizzaGenerator" repository.

What I Learned From this Project

This project was a good jumping off point for learning about web development. I learned how to use the React.Js framework, how to instantiate and test a web extension in Google Chrome using JSON, and how to link different code files together; something that I have always wondered and did not have much experience with from my time in school. I got the opportunity to re-learn HTML and CSS, and also got the opportunity to actually build something with JavaScript. This first project I believe is a good exercise in learning and understanding what front end development looks like. I had a lot of fun with this project and am looking forward to doing more things like this.

How to Make This Project Better

As of right now, I am still working on having the extension communicate with dominos.com to automatically place the order for you once approved. I would like to add a surprise function which would place the order and not tell you what the pizza is until you receive the pizza.
